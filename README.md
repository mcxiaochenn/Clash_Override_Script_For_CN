# Clash Override Script For CN

适用于 [mihomo (Clash Meta)](https://github.com/MetaCubeX/mihomo) 的覆写脚本，提供国内分流规则优化。

## 功能

- 国内域名/IP 直连分流规则
- 适配 mihomo 内核覆写格式

## 使用方法

1. 打开你的 mihomo 客户端
2. 在覆写（Override）配置中导入本脚本
3. 根据需要修改规则或代理组
